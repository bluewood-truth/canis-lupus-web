SW마에스트로 제 12기 B2K팀의 모임 관리 서비스 '이건모임'의 Web Front-end 개발 Repository입니다.

- [React.js](https://reactjs.org/)

<br>

## Installation & Deployment

1. 저장소를 clone합니다.

   ```
   git clone https://github.com/your_username_/Project-Name.git
   ```

2. NPM 패키지들을 설치합니다.

   ```
   npm install
   ```

   또는

   ```
   yarn
   ```

3. 프로젝트를 빌드하고 실행합니다.

   ```
   npm run build
   npm start
   ```

   또는

   ```
   yarn build
   yarn start
   ```
